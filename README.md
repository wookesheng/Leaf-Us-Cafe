# ☕ Leaf Us Cafe
A modern cafe website designed to showcase menu, branding, and user experience.

---

## 📖 Overview
Leaf Us Cafe is a responsive website created to represent a cafe's online presence. It highlights menu offerings, branding, and a clean user interface to enhance customer experience.

This group project was developed as a portfolio piece to demonstrate frontend development and UI/UX design skills.

---

## ✨ Features
- 📋 Interactive menu display
- 🖼 Visually appealing layout and design
- 📱 Responsive design for desktop and mobile
- 🧭 Easy navigation between sections
- 🎨 Consistent branding and styling
